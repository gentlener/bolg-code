一、什么是 vdom

  • 用 JS 模拟 DOM 结构
  • DOM 变化的对比，放在 JS 层来做
  • 提高重绘性能

遇到的问题

  • DOM 操作是“昂贵”的，js 运行效率高
  • 尽量减少 DOM 操作，而不是“推倒重来”
  • 项目越复杂，影响就越严重
  • vdom 即可解决这个问题
